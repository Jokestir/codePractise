Get path apis
==============


os.path.abspath(fileName)                      : full path of the file including filename

# os.path.abspath(__file__)  will return D:/example.py

os.getcwd()                                     : get current working directory.

# os.path.getcwd() will return C:\users\chirag.ahuja


Split path names
================

os.path.dirname(fileName)                      : the directory. full path name excluding the last backslash

# to get the directory of the file. os.path.dirname(os.abspath(__file)) will return D:\


os.path.basename(fileName)                      : return only the string after the last backslash.

# to find the filename. os.path.basename(os.getcwd()) will return  chirag.ahuja


os.path.splitext(file)                           : list containing the extension and rest of the path

#  find the extension of a file


Join path names
===============


os.path.join(folder1,folder2) : joins path intellligently 

usage : to create a new file at the current location or to combine paths in platfornm independent manner


Explore contents
=================


os.listdir(path)      : list all the contents in the path

os.path.isfile(path)  : returns True if path if a file

os.path.isdir(path)   : returns False if the path is a folder

os.path.exists(path)  : returns True if the path exits
